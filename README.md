# Hackathon da FEA.dev 🚀

Bem-vindo ao Hackathon de Desafios de Estrutura de Dados e Algoritmos da FEA.dev! Este evento é uma oportunidade emocionante para os membros da entidade estudantil da FEA USP mostrarem suas habilidades em programação.

### Sobre o Evento
- Data: 09/03/2024 (12:30 - 20:00)  
- Local: Cosmos USP


Bem-vindos, caros devers! Vocês serão divididos em equipes para enfrentar os desafios mais complexos da computação. Juntos, devem desvendar enigmas e superar obstáculos para testar seu conhecimento. 😎

Lembrem-se, não basta apenas chegar ao resultado final; o segredo está em encontrar o caminho mais eficiente e rápido para resolver os desafios. A equipe que conceber a melhor estratégia será coroada como vencedora! 🏆

Testem seus códigos à vontade, mas saibam que uma vez submetidos, não poderão voltar atrás. O destino aguarda e em breve receberão o veredito!🤯

Que os bits estejam com você! 🌟👩‍💻🧙‍♂️
